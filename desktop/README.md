# Smart Attendance Desktop App

## Quick Start

### Install Dependencies
```bash
cd desktop
npm install
```

### Run Desktop App
```bash
npm start
```

The app will:
1. Automatically start the Python backend server
2. Open a desktop window with the interface
3. You can upload images and analyze engagement directly

### Build Installer (Optional)
```bash
npm run build
```

This creates a Windows `.exe` installer in `desktop/dist/`

## Features
- ✅ No browser needed
- ✅ Auto-starts backend
- ✅ Native desktop experience
- ✅ Clean UI without browser chrome
